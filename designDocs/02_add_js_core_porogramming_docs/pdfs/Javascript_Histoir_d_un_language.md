## Reseaux (Internet)

_Définition_

```
1. « réseau international de communication entre utilisateurs d’ordinateurs »
```
2. Internet est le réseau informatique mondial qui rend accessibles au public des services variés
comme le courrier électronique, la messagerie instantanée et le World Wide Web, en utilisant le proto-
cole de communication IP (internet protocol). ... ( _wikipedia_ )
3. Réseau des réseaux à l’échelle mondiale qui effectue l’interconnexion d’un grand nombre de réseaux
internationaux, régionaux et locaux, tous basés sur un jeu de protocoles commun à savoir TCP/IP

Définie par conjugaison de 5 critères :
a. distribuée et incarnée dans de multiples structures
b. transparente (pour l’utilisateur)
c. de dimension collective
d. étendue et transversale
e. normalisée

Réseaux :
a. Internetworking, réseau de réseaux
b. Protocoles :
TCP/IP, HTTP, FTP, SMTP...
c. Applications :
Messagerie, forums, WWW, chat, etc.
d. Services :
Communication, recherche d’information, publications...
e. Ressources :
Sites web, blogs, bibliothèques numériques...
