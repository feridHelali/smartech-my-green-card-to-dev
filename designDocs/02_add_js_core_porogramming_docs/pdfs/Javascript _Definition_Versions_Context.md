# Javascript : Définition - Versions - Context

## Définition

JavaScript est un langage de programmation de scripts principalement employé dans les pages web inter-
actives et à ce titre est une partie essentielle des applications web. Avec les technologies HTML et CSS, Ja-
vaScript est parfois considéré comme l’une des technologies cœur du World Wide Web2. Une grande ma-
jorité des sites web l’utilisent, et la majorité des navigateurs web disposent d’un moteur JavaScript dédié
pour l’interpréter, indépendamment des considérations de sécurité qui peuvent se poser le cas échéant. |

Le langage a été créé en dix jours en mai 1995 pour le compte de la Netscape Communications Corporation
par Brendan Eich. ( _wikipedia_ )

## Context


## Client Side Scripting (Coté Client)

Le code côté client est du code exécuté sur l’ordinateur de l’utilisateur : quand une page web est vue, le
code côté client de la page est téléchargé, puis exécuté et affiché par le navigateur. Dans ce module JavaS-
cript, nous parlons explicitement de JavaScript côté client.

## Server Side Scripting (coté Serveur)

Le script côté serveur est une technique utilisée pour développer des sites Internet avec des éléments dy-
namiques et des applications Web. Elle est basée sur l’utilisation de scripts, qui fonctionnent sur le serveur
Web en utilisant Javscript ( **Nodejs** ) lorsqu’un client demande le contenu correspondant ( _ressource_ ).

https://www.ionos.fr/digitalguide/sites-internet/developpement-web/les-differences-entre-les-scripts-
cote-serveur-et-cote-client/

## Mode d’éxécution

```
A. Javascript Engine (Moteur Javascript)
Un moteur JavaScript est un programme logiciel qui interprète et exécute du code en lan-
gage JavaScript. Les moteurs JavaScript sont généralement intégrés aux navigateurs Web.
Le premier moteur JavaScript, SpiderMonkey, a été créé par l’informaticien américain
Brendan Eich pour le navigateur Netscape Navigator. Il était programmé en langage C.
Les premiers moteurs JavaScript étaient de simples interpréteurs. Puis, les versions ont
évoluées et ont intégrés la compilation afin d’améliorer les performances d’exécution du
code JavaScript.
```
